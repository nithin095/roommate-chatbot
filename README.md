# 👥 Roommates Assistant Chatbot

A simple and interactive **Flask-based chatbot** to manage roommate activities like tasks, rent tracking, and cleaning responsibilities — with voice support.

---

## 📌 Features

* 🧑‍🤝‍🧑 View all roommates
* 📋 Task management for each member
* 💰 Rent status tracking
* 🧹 Cleaning guidance
* 🎤 Voice input (speech recognition)
* 🔊 Voice output (text-to-speech)
* 💬 Modern chat UI

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, JavaScript
* **Browser APIs:** SpeechRecognition, SpeechSynthesis

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/roommate-chatbot.git
cd roommate-chatbot
```

### 2. Install dependencies

```bash
pip install flask
```

### 3. Run the application

```bash
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000/
```
